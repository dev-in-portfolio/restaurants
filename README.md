# Restaurant Directory Portal

A premium, highly aesthetic multi-page restaurant directory and portal built with vanilla HTML5, modern CSS3, and interactive ES6 JavaScript.

## Features

- **Central Portal (`index.html`)**: A clean navigation directory page displaying curated restaurants with ratings, cuisine tags, pricing levels, and animated routing buttons to each restaurant's dedicated page.
- **Gourmet Bistro (`gourmet-bistro.html`)**: A warm, classy serif-based page designed for a fine French-American dining establishment. Features a detailed, clean food menu and a table reservation system.
- **Sushi Zen (`sushi-zen.html`)**: A modern, sleek dark-themed minimalist website for an authentic Japanese sushi bar. Includes a visual signature roll showcase and reservation capabilities.
- **La Piazza (`la-piazza.html`)**: A cozy, terracotta-hued layout for a traditional Italian pizzeria and trattoria. Features categorized menu tabs and booking capabilities.
- **Interactive Mock Reservations**: Forms on each page capture bookings and present micro-animations for success feedback.
- **Responsive Layout**: Fluid grids and flexboxes that adapt seamlessly from mobile devices to desktop monitors.

## Getting Started

1. Clone or download this repository.
2. Open `index.html` in any web browser to view the portal.
3. Click "Visit Website" on any restaurant card to navigate to its dedicated page.
